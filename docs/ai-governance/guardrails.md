# AI Governance and Guardrails

## Non-Negotiable Rule

The LLM cannot execute access changes.

## Controls

- RAG-grounded recommendations
- Required policy citations
- Structured JSON schema
- Confidence threshold
- Human approval before mutation
- Prompt-injection filtering
- PII minimization in prompts and logs
- LLM output validation
- Evaluation metrics for grounding, latency, cost, and correctness
